# Rack::SimpleAuth

Rack Middleware for HMAC Authentication

## Installation

Add this line to your application's Gemfile:

    gem 'rack-simple_auth'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rack-simple_auth


## Usage

Uses ```X-Content-Hash``` and ```X-Public-Key``` Headers
